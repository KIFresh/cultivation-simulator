# 战斗系统 — 设计规范

## 概述

为修仙模拟器增加 PVE 战斗系统：行动和奇遇可触发战斗，基于玩家属性/功法/装备计算战力权重，随机判定胜负，差异化叙事配合四段式战败惩罚，完整战利品掉落和功法残页合成闭环。

---

## 1. 战力计算公式

```
基础战力 = max(10, 根骨×2 + 灵性×1.5 + 悟性×1)    // 保底 10
境界系数 = 1.5 ^ 境界索引
  (凡人=1, 炼气=1.5, 筑基=2.25, 结丹=3.375, 元婴=5.0625,
   化神=7.594, 炼虚=11.39, 合体=17.09, 大乘=25.63, 渡劫=38.44)
功法加成 = 1 + (装备功法的 combat 值之和 / 100)
装备加成 = 1 + (装备物品的 combat 值之和 / 100)

玩家战力 = 基础战力 × 境界系数 × 功法加成 × 装备加成
敌人战力 = 敌人预设值
```

### 1.1 装备 combatValue 数值

| 类型 | 范围 | 示例 |
|------|------|------|
| 武器 | 5~20 | 木剑+5, 铁剑+8, 灵剑+15 |
| 防具 | 3~10 | 布衣+3, 皮甲+5, 法袍+8 |
| 饰品 | 2~8 | 玉佩+3, 灵珠手串+5 |
| 法宝 | 10~30 | 待定 |

## 2. 胜负判定

```typescript
// 绝对碾压保护
if (玩家战力 >= 敌人战力 × 5) → 直接胜利

// 正常权重判定
胜率 = 玩家战力 / (玩家战力 + 敌人战力)
随机数 < 胜率 → 胜利
随机数 ≥ 胜率 → 失败
```

| 胜率区间 | 胜负 | 叙事风格 | 文案示例 |
|---------|------|---------|---------|
| ≥ 70% | 胜利 | 轻松碾压 | "你随手一挥，剑气纵横，妖兽灰飞烟灭" |

## 4. 战败惩罚

```
ratio = 敌人战力 / 玩家战力
```

| 档 | 条件 | 惩罚 | 叙事 |
|----|------|------|------|
| 0 | ratio < 1 | 丢 5~10% 灵石 + debuff「道心受损」1次（悟性-2） | 喜剧翻车 |
| 1 | 1~2x | 丢 20~50% 灵石 + 丢 1~2 件非装备物品 | 惜败 |
| 2 | 2~5x | 丢 50~80% 灵石 + 重伤 debuff 3次（属性减半）+ 扣 5~10 年寿元 | 惨败 |
| 3 | 5x+ | 直接触发道消事件 | 身死道消 |

### 4.1 重伤 Debuff

- 字段：`injuryDebuff Int @default(0)`（剩余行动次数）
- 受伤赋值 3，每次行动消耗 1
- `injuryDebuff > 0` 时：`calculateActionExp` 结果 × 0.5
- 可使用愈灵符解除

## 5. 战利品掉落

### 5.1 常规掉落

```
灵石 = floor(敌人战力 × (0.5 + random × 1.0))
修炼值 = floor(敌人战力 × (2 + random × 1))
```

### 5.2 稀有掉落（受气运影响）

```typescript
基础爆率 = 普通 0.05 / 精英 0.15 / BOSS 0.30
气运修正 = 1 + 气运 × 0.1
最终爆率 = 基础爆率 × 气运修正

// Loot Explosion：超过 100% 的部分转化为额外判定
function resolveDrops(rate: number): number {
  let drops = 0;
  while (rate > 0) {
    if (Math.random() < rate) drops += 1;
    rate -= 1;
  }
  return drops;
}
```

### 5.3 越阶奖励

```
境界差 = 敌境界索引 - 玩家境界索引
越阶爆率 = 境界差 × 0.1
额外掉落：高一阶材料或功法残页
```

## 9. 触发流程

### 9.1 行动触发

```
玩家执行行动（如「外出历练」）
  → 30% 概率触发战斗检查
  → 今日战斗次数 < 5？
    → 否：正常行动流程
    → 是：调战斗引擎
      → 选择敌人（按地点+境界）
      → 计算战力 → 判定胜负
      → 战斗奖励替换原有 exp
      → 应用惩罚 / 发放战利品
      → 生成战斗叙事（替代行动叙事）
      → 保存 GameEvent (type="COMBAT")
      → 返回战斗结果 + 叙事
```

### 9.2 奇遇触发

```
奇遇高风险选项 → 调战斗引擎（传 enemyId）
  → 走战斗流程
  → 胜利 → 原奖励 + 战利品
  → 失败 → 走惩罚档位
```
