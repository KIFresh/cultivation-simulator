# 功法系统 — 设计规范

## 概述

为修仙模拟器增加功法系统：玩家可通过多种途径收集功法，装备后获得被动加成（修炼速度、突破概率、战斗属性、生活效果）。功法可升级，品级从凡到天共五阶。

---

## 1. 数据模型

### 1.1 Prisma — 新增 CultivatorTechnique 表

```prisma
model Cultivator {
  // ...现有字段不变...
  techniques    CultivatorTechnique[]
}

model CultivatorTechnique {
  id            String   @id @default(cuid())
  cultivatorId  String
  cultivator    Cultivator @relation(fields: [cultivatorId], references: [id])
  techniqueId   String
  equipSlot     Int?       // 数字=装备槽位（1/2/3），null=未装备
  level         Int        @default(1)
  proficiency   Int        @default(0)
  createdAt     DateTime   @default(now())

  @@unique([cultivatorId, techniqueId])
  @@index([cultivatorId])
}
```

### 1.2 TypeScript — 功法数据定义

```typescript
/** 功法品级 */
type TechniqueGrade = "凡" | "黄" | "玄" | "地" | "天";

/** 可装备的境界 —— 严格联合类型 */
type TechniqueRealm = "凡人" | "炼气期" | "筑基期" | "金丹期" | "元婴期" | "化神期" | "炼虚期" | "合体期" | "大乘期" | "渡劫期";

/** 效果类型 */
type EffectType = "cultivationSpeed" | "breakthroughRate" | "combat" | "daily";

/** 单个效果 */
interface TechniqueEffect {
  type: EffectType;
  value: number;        // 基础值（1级时）
  perLevel: number;     // 每升1级增加值
  description: string;  // 显示文本，如 "修炼速度 +{value}%"
}

/** 功法定义 */
interface Technique {
  id: string;
  name: string;
  icon: string;
  description: string;
  grade: TechniqueGrade;
  realm: TechniqueRealm;          // 最低境界要求
  maxLevel: number;               // 最高等级（默认3）
  upgradeProficiency: number[];   // 升级所需熟练度，如 [100, 300] 表示 2级需100，3级需300
  effects: TechniqueEffect[];
}
```

## 2. 核心函数

### 2.1 获取功法效果

```typescript
/** 计算装备功法的总加成 */
export function calculateTechniqueBonuses(
  techniques: { technique: Technique; level: number }[]
): Record<EffectType, number> {
  const result: Record<string, number> = { cultivationSpeed: 0, breakthroughRate: 0, combat: 0, daily: 0 };
  for (const t of techniques) {
    for (const e of t.technique.effects) {
      result[e.type] = (result[e.type] || 0) + e.value + e.perLevel * (t.level - 1);
    }
  }
  return result;
}
```

### 2.2 熟练度与升级

```typescript
/** 增加功法熟练度，返回是否升级 */
export function addProficiency(
  currentLevel: number,
  currentProf: number,
  upgradeProficiency: number[],
  amount: number
): { newLevel: number; newProficiency: number; leveledUp: boolean } {
  const newProf = currentProf + amount;
  const maxLevel = upgradeProficiency.length + 1;
  if (currentLevel >= maxLevel) {
    return { newLevel: currentLevel, newProficiency: currentProf, leveledUp: false };
  }
  const required = upgradeProficiency[currentLevel - 1];
  if (newProf >= required) {
    return { newLevel: currentLevel + 1, newProficiency: newProf - required, leveledUp: true };
  }
  return { newLevel: currentLevel, newProficiency: newProf, leveledUp: false };
}
```

## 3. 获取方式

| 途径 | 说明 |
|------|------|
| 初始赠送 | 创建角色时自动获得「吐纳术」 |
| 商店购买 | 坊市可购买基础/黄级功法 |
| 奇遇奖励 | 随机奇遇可选择获得功法 |
| 突破奖励 | 首次突破至新境界时随机解锁对应功法 |
| 年志奖励 | 特定年龄/事件触发获得 |

## 5. API

### 5.1 读取功法列表

`GET /api/cultivator/techniques?userId=X`

```json
{
  "techniques": [{ "techniqueId": "basic_breathing", "equipSlot": 1, "level": 2, "proficiency": 150 }],
  "allTechniques": { /* TECHNIQUES 常量数据 */ }
}
```

### 5.2 装备/卸下功法

`POST /api/cultivator/techniques`

```json
{ "action": "equip", "userId": "xxx", "techniqueId": "sword_foundation", "slot": 2 }
{ "action": "unequip", "userId": "xxx", "techniqueId": "sword_foundation" }
```
